# task-rustler-cli
Learning Rust By Building A CLI To Add Tasks

### Commands
- `cargo run list`
- `cargo run add [TASK_NAME]`
- `cargo run remove [TASK_NAME]`
- `cargo run clear`
- `cargo run sort`


### V1 Features
- [x] List Tasks
- [x] Add Task
- [ ] Complete Task
- [x] Remove task
- [x] Sort Tasks
- [x] Clear Tasks


### V2 Features
- [ ] GUI Via [Ratatui](https://ratatui.rs/)

### V3 Features
