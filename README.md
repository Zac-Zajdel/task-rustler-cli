# task-rustler-cli
Learning Rust By Building A CLI To Add Tasks

### Commands
- `cargo run list`
- `cargo run add [TASK]`
- `cargo run clear`


### V1 Features
- [x] List Tasks
- [x] Add Task
- [ ] Complete Task
- [ ] Remove task
- [ ] Sort Tasks
- [x] Clear Tasks


### V2 Features
- [ ] GUI Via [Ratatui](https://ratatui.rs/)

### V3 Features
- [ ] Read and Save To Postgres Database
